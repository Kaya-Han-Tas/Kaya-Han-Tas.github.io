An Earth-Sized Planet in a 5.4h Orbit Around a Nearby K dwarf

Worked on the discovery paper of an Ultra-Short Period exoplanet, TOI-2431 b, as the first author of the paper, with researchers in NASA and various other institutes. The paper is live on arXiv and is expected to be published in the Astronomy & Astrophysics (A&A) journal in the near future.

  • Paper - arXiv

ExoMUSE - A tool for Exoplanetary Analysis

Developed ExoMUSE (Exoplanet Modelling and Understanding for Science and Education) as my Master's thesis. The tool has the ability to do Radial Velocity and Transit analysis, alongside a future implementation for Astrometry (WIP). It has also been validated on the planet TOI-2431 b.

  • ExoMUSE - GitHub

La Palma - Observation Project

Proposed the project for the exoplanet candidate, TOI-2431.01, and secured telescope time at the Mercator Telescope, La Palma as a part of a team of three MSc students. Conducted observations in La Palma for 5 nights, resulting in the discovery of an Ultra-Short Period (USP) exoplanet, TOI-2431 b. Below are some fun highlights from there.

Bachelor's Thesis - Light Curve Modelling

Wrote a Python code that can do light curve analysis using Least-Squares method. The graphical user interface (GUI) and the Radial Velocity method was added to the code after my graduation as a side project.
